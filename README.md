## Feature
- Single Page Application
- RESTful API
- Redux Store (Redux-Saga)
- Testing

## Setup

```sh
$ git clone git@github.com:ardamutlu/todo-list-web.git
$ cd todo-list-web
$ npm i
```

## Development

```sh
Client
$ npm start
$ open http://localhost:4000

Server
$ npm run server:dev
$ open http://localhost:3000
```

## Test

```sh
$ npm test
```

## Production

```sh
$ npm run build 
```
